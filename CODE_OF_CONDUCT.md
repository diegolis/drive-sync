# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its Code of Conduct. By participating in this project — submitting issues, pull requests, comments, or any other interaction — you agree to abide by its terms.

The full text is available at the link above and on the [Contributor Covenant website](https://www.contributor-covenant.org/).

## Reporting

If you experience or witness a violation, contact the project maintainer through the email address listed on the maintainer's GitHub profile, or open a private GitHub Security Advisory if you would prefer a tracked channel.

Reports will be handled confidentially. Retaliation against reporters is itself a violation.

## Scope

This Code of Conduct applies to all project spaces — the repository, issue tracker, pull requests, discussions, and any community channels associated with the project — and to public spaces when an individual is representing the project.
